## How to use;

### Install a dependency module.

    $ cpanm Mojolicious

### Get talk lists from the YAPC::Asia web site.

    $ perl ./get_talks.pl

Then, `talks.json` file is generated.

### Open the HTML with talk lists. index.html(Angular.js),index2.html(Vue.js)

    $ open index.html
    $ open index2.html

Enjoy!
